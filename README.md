# DeclaraIMCED
Sistema de ayuda para la declaración en SAT MICH.


El correcto funcionamiento del sisitema requiere:

-PHP 8.2+
-Bootstrap 5.3.8
-bootstrap iconos 1.13.1
-fpdf

*Para modificar valores, localiza el archivo valores.php de raiz y actualiza la información.

Se puede correr en Local sin problemas, no requiere base de datos ni complementos adicionales.
